# Lesson 3.17 Flexbox Container

Example style sheet:
```
<style type="text/css">
  .container {
    width: 100%;
    display: flex;
    flex-wrap: wrap;
  }
</style
```

`display: flex;` allows the elements to expand and shrink to fit the browswer display.

`flex-wrap: wrap;` allows the elements to wrap to the next row.
