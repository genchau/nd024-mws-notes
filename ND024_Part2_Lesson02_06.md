# Lesson 2.6 Quiz: Calculating CSS Pixels

##### Quiz:
Given a screen of 1920 by 1080 pixels,
and a device pixel ratio of two. What is the maximum width of the viewport in landscape mode measured in CSS pixels?
##### Solution: 
960 CSS Pixels
