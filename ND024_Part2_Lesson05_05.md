# Lesson 5.5 Quiz: Hide Some Columns

##### Quiz: 
- gametime class is hidden
- for viewports 499px wide or smaller

##### Solution:
```
@media screen and (max-width: 499px) {
  .gametime {
    display: none;
  }
}
```
